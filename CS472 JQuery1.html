1. In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?
The jQuery function is a factory for creating a jQuery object that represents one or more DOM nodes. jQuery objects have methods to manipulate these nodes. These methods (sometimes called commands), are chainable as each method also returns a jQuery object.
2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red. Sample HTML: 
                    <body>
                    <h1>abc</h1><br><div>
                    <h1>div-1</h1> <h1>div-2</h1> </div> <h1>xyz</h1>
                </body>
                
$('div h1').css({'background': 'red'});
